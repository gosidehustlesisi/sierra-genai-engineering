## What I Build

I build **retrieval-augmented generation systems** that process live academic and legal corpora. Every document in this portfolio was fetched from a real API — arXiv, PubMed, SCOTUS, or Wikipedia — not fabricated. Every embedding was computed. Every classification was trained on real text.

This is not a tutorial repo. This is a **production engineering portfolio** for GenAI and NLP systems.
